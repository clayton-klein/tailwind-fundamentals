Styling based on parent state (group):

In this example the idea is to change the color of the texts (h3 and p) inside the link when we hover over the card (and not only when we hover over the texts themselves), to do it we added the class "group" in the parent element (a) and then we use "group-" before the hover state in the children elements (h3 and p).

Pseudo Classes:

Even though we set the "first:" pseudo class in all "li", the style is apllied only on the real first one.

To change the bg color of every other "li" we had to add "even:" and "odd:" pseudo class in all of the items and then it was applied accordingly.
