* By default, we can have our project use whatever mode (dark or light) our OS is using.

* We can use the prefix "dark:{class}" to set some styles only for "dark-mode".

* We can also set Tailwind config to use a class instead of using our OS mode (we actually did it in this case, but just for the toggle btn, inspect the page to see the config and style in the head and the script at the bottom of the page).
