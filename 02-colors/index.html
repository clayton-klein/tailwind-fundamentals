Box Shadow Colors:

Opacity defaults to 100, but we can change it with a slash "/" after the "shade" number of the color.

Arbitrary Colors:

We can put any value we want between [...] after the class name, this way we are not stuck on the default values of the framework.

bg-[#5942DF] bg-[rgb(255,0,0)] bg-[steelblue]
