Tailwind is mobile-first!

Resize the window to see the background-color changes...

To make changes based on the screen size we just need to prefix our classes with the size of the screen we want to attend...
(sm = 640px, md = 768px, lg = 1024px, xl = 1280px, 2xl = 1536px)
...followed by a colon (without spaces).

class applied to the body: class="bg-black sm:bg-green-800 md:bg-yellow-700 lg:bg-blue-500 xl:bg-pink-500 2xl:bg-red-500 mb-8 mx-8"
